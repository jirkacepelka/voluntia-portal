# voluntia-portal

Statická HTML/CSS verze stránky Voluntia portálu (bez build nástrojů a bez frameworků).

## Struktura

- `index.html` – hlavní entrypoint
- `css/kulaty_stul.css` – styly stránky

## Spuštění lokálně

1. Stáhni repozitář jako ZIP (nebo klonuj).
2. Rozbal archiv.
3. Otevři soubor `index.html` v prohlížeči (funguje přes `file://`).

## Volitelně: jednoduchý lokální server

Pokud chceš spouštět stránku přes `http://` místo `file://`, můžeš použít například:

```bash
python3 -m http.server 8000
```

Pak otevři `http://localhost:8000`.
